*RAG CHATBOT USING LANGCHAIN & STREAMLIT*

A simple Retrieval-Augmented Generation (RAG) chatbot built with LangChain and Streamlit that allows users to upload documents and ask questions. The system retrieves relevant chunks from the documents and generates accurate answers using a Large Language Model.

*FEATURES*

Upload text or PDF documents

Automatic text chunking & embeddings

Vector database storage (FAISS / Chroma)

Semantic search over documents

Conversational chat interface

Fast and lightweight Streamlit UI

*TECH STACK*

Python

LangChain

Streamlit

FAISS or Chroma

HuggingFace / OpenAI / Groq LLM

Sentence Transformers
